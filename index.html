03 — Data Flow
Processing Pipeline
Five stages from raw downlink to geo-locked result in under 50ms.
01
📁 File Drop → fsnotify Trigger
Raw EO imagery lands in ./downlink_buffer/. The Go satellite service detects the CREATE event within microseconds via Linux kernel inotify hooks.
02
☁️ Dual-Upload: MinIO + Kafka
File is streamed to satellite-raw MinIO bucket via zero-copy multipart. A Kafka event is produced to eo-events topic with object key metadata.
03
🧠 ViT Inference + Cosine Search
Python worker fetches the image from MinIO, runs ViT-B/16 forward pass to get a 768-dim embedding, then cosine similarity search against the 874-tile Sentinel-2 corpus finds the nearest geographic location.
04
🗺️ PostGIS Write + Atomic File Sync
Results (lat/lon, confidence, tile_id, processing_ms) are inserted into PostGIS with GIST geometry index. Processed image is atomically written to local_sync/latest_processed.jpg.
05
📡 Live Telemetry Fan-out
TUI Dashboard polls PostGIS every 100ms. Raylib GUI polls file mod-time for instant image refresh. Prometheus scrapes all services. Grafana renders pre-built dashboards.
